## Installation
Clone the repository: 
```bash
git clone --recurse-submodules git@github.com:AyonRRahman/Thesis.git
```
## Download the Eiffel Tower dataset
```bash
cd data
chmod +x datasetup.sh
./datasetup.sh
```

## Download the Kitty Raw dataset
```bash
cd data
./Kitty_raw_data_downloader.sh
```
